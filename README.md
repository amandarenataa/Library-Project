# 📚 Library Management System
### ALP Revisi — Semester 2 Project
 
A desktop-based Library Management System built with **C# (.NET)** and **Windows Forms**, developed as part of the Algorithm and Programming course at Universitas Ciputra Surabaya.
 
---
 
## 📋 Overview
 
This application provides a complete library management solution, covering the full lifecycle of book borrowing — from member registration and book cataloguing to borrowing, returning, and receipt generation.
 
---
 
## ✨ Features
 
| Feature | Description |
|---|---|
| 📖 Add Book | Register new books into the library catalogue |
| ✏️ Edit Book | Update existing book information |
| 🗑️ Remove Book | Delete books from the system |
| 👤 Register Member | Add new library members |
| 📤 Borrow Book | Record book borrowing transactions |
| 📥 Return Book | Process book returns |
| 🧾 Nota Totalan | Generate transaction receipts and summaries |

## 🚀 How to Run
 
1. Open **Visual Studio**
2. Click **File → Open → Project/Solution**
3. Select `ALP Revisi (2).csproj`
4. Press **F5** or click **Start** to run the application
 
> **Requirements:** .NET Framework installed, Visual Studio 2019 or later recommended.
